# React Starter

Clone the repository

```bash
mkdir -p ~/GitHub/rkiel
cd ~/GitHub/rkiel
git clone git@github.com:rkiel/react-starter.git  NEW_PROJECT
cd NEW_PROJECT
```

Make it your own

```bash
rm -rf .git
git init
git add .
git commit -m "Initial commit"
```

Install packages

```bash
yarn install
```

Start the server

```bash
yarn start
```

View in browser

[http://localhost:3000](http://localhost:3000)
